# Project Name

Simple SPA Movie Web App

## Features

- Get a random movie quote
- Get featured movies
- Search for movie
- View details about movie

## Technologies Used

- Javascript
- Vite
- React
- React Forms
- Yup
- Axios
- Sass

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/xHacka/React.git
   cd React
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run project:
    ```bash
    npm run dev
    ```
